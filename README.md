# Equine Gait & Heart Rate Analysis 🐎📉

A research project analyzing the biomechanics of walk, trot, and canter gaits in horses using wireless Inertial Measurement Units (IMUs) and heart rate monitors to objectively detect lameness and monitor rehabilitation.

## 📌 Project Overview
Lameness in horses, often caused by orthopedic pain, is traditionally assessed through subjective visual examinations. This project provides an objective approach to diagnosing lameness by:
* Using the EquiMoves system to capture high-precision 3D motion data.
* Analyzing walk, trot, and canter gaits across various breeds to identify asymmetries resulting from past injuries.
* Exploring the correlation between heart rate and moments of pain during movement using a Polar V800 monitor.
* Synchronizing multi-sensor time-series data for meaningful physiological and biomechanical analysis.

## 🔬 Methodology
* **Data Collection:** Horses were lunged at various gaits and directions in an indoor arena.
* **Motion Tracking:** Seven wireless, time-synced IMUs (precision: 100 ns) were placed on key body points to capture 3D motion.
* **Physiological Monitoring:** A Polar V800 heart rate monitor (chest strap and receiver) was utilized in select sessions to collect heart rate data.
* **Data Processing:** All sensor streams were synchronized via time and stride for integrated analysis.
